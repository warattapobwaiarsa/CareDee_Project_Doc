💡 คำอธิบายสถานะและการจัดการเงิน (Status Glossary)

ความหมายของสถานะ เกตเวย์

  • รอดำเนินการ : รออนุมัติ - รายการจองเสร็จสิ้นแล้ว รอ Admin ตรวจสอบและกด "อนุมัติ" เพื่อเริ่มกระบวนการโอนเงิน
  • กำลังดำเนินการ : กำลังโอน - ระบบส่งคำสั่งไปยังธนาคารแล้ว อยู่ระหว่างรอผลตอบรับ (ไม่สามารถยกเลิกได้ในขั้นตอนนี้)
  • สำเร็จ : โอนสำเร็จ - เงินเข้าบัญชีผู้รับปลายทางเรียบร้อยแล้ว
  • ล้มเหลว : ล้มเหลว - การโอนไม่สำเร็จ (เช่น เลขบัญชีผิด, ยอดเงินในระบบไม่พอ) ต้องตรวจสอบและแก้ไขข้อมูล

การดำเนินการ (ความหมายของการดำเนินการ)

  • อนุมัติ (อนุมัติ): ตรวจสอบยอดเงินและความถูกต้องเพื่อเปลี่ยนสถานะเป็นพร้อมโอน
  • โอนเงิน (โอนเงิน): ส่งคำสั่ง Batch Payout ไปยัง ระบบรับชำระเงิน ทันที
  • ตรวจสอบบัญชี (ตรวจสอบบัญชี): ตรวจสอบความถูกต้องของชื่อ-เลขบัญชีกับข้อมูลที่ลงทะเบียนไว้
  • ดูรายละเอียด: ดูรายละเอียดการหัก Commission, ภาษี ณ ที่จ่าย และยอดสุทธิที่โอนจริง
